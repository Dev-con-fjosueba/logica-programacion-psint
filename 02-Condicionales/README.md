# 02 - Condicionales

## Objetivo

Practicar el uso de estructuras condicionales para tomar decisiones dentro de un algoritmo según determinadas condiciones.

---

## Conceptos Trabajados

- Estructura `Si`
- Estructura `Si-Sino`
- Condiciones compuestas
- Operadores relacionales
- Operadores lógicos
- Toma de decisiones

---

## Competencia Desarrollada

Aplicar estructuras de decisión para resolver problemas que requieren evaluar condiciones y ejecutar diferentes acciones según el resultado obtenido.

---

## Ejercicios

### C01 - Número Par o Impar

**Descripción:**
Determinar si un número entero ingresado por el usuario es par o impar utilizando el operador módulo.

**Conceptos:**

- Operador MOD
- Estructura Si-Sino

---

### C02 - Mayor de Tres Números

**Descripción:**
Solicitar tres números y determinar cuál de ellos posee el valor más alto.

**Conceptos:**

- Comparaciones
- Condiciones compuestas
- Estructuras condicionales

---

### C03 - Aprobado o Desaprobado

**Descripción:**
Determinar si un estudiante aprueba o desaprueba un curso según su nota final.

**Criterio de Evaluación:**

- Nota ≥ 11 → Aprobado
- Nota < 11 → Desaprobado

**Conceptos:**

- Comparación de valores
- Toma de decisiones

---

### C04 - Descuento en Tienda

**Descripción:**
Calcular el descuento aplicable a una compra según el monto total.

**Reglas:**

- Compra ≥ S/ 200 → 15% de descuento
- Compra < S/ 200 → 5% de descuento

**Conceptos:**

- Porcentajes
- Condicionales
- Operaciones aritméticas

---

### C05 - Calculadora Básica

**Descripción:**
Permitir al usuario seleccionar una operación matemática y mostrar el resultado correspondiente.

**Operaciones Disponibles:**

1. Suma
2. Resta
3. Multiplicación
4. División

**Conceptos:**

- Menús
- Condicionales múltiples
- Validación de datos

---

## Aprendizajes Esperados

Al finalizar esta sección seré capaz de:

- Resolver problemas mediante estructuras de decisión.
- Analizar diferentes escenarios dentro de un algoritmo.
- Implementar lógica condicional para controlar el flujo de ejecución.
- Diseñar soluciones más dinámicas y adaptables.
