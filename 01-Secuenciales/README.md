# Secuenciales

## Objetivo

Practicar algoritmos donde las instrucciones se ejecutan
de manera lineal, sin estructuras de decisión ni repetición.

## Conceptos

- Variables
- Constantes
- Operadores aritméticos
- Entrada de datos
- Salida de datos

## Ejercicios

1. Área de un triángulo
2. Promedio de notas
3. Conversión de temperatura
4. Salario semanal
5. Distancia recorrida
