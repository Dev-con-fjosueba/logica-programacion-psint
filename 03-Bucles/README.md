# 03 - Bucles

## Objetivo

Practicar el uso de estructuras repetitivas para automatizar tareas, procesar datos y resolver problemas que requieren ejecutar un conjunto de instrucciones múltiples veces.

---

## Conceptos Trabajados

- Bucle `Para`
- Bucle `Mientras`
- Bucle `Repetir Hasta Que`
- Contadores
- Acumuladores
- Variables de control
- Ciclos controlados por condición
- Ciclos controlados por contador

---

## Competencia Desarrollada

Implementar estructuras repetitivas para automatizar procesos y resolver problemas que requieren ejecutar una misma acción múltiples veces.

---

## Ejercicios

### B01 - Tabla de Multiplicar

**Descripción:**

Solicitar un número entero y mostrar su tabla de multiplicar.

**Conceptos:**

- Bucle Para
- Contadores
- Multiplicación

---

### B02 - Suma de los Primeros N Números

**Descripción:**

Calcular la suma de los primeros N números naturales.

**Conceptos:**

- Acumuladores
- Bucle Para
- Series numéricas

---

### B03 - Factorial de un Número

**Descripción:**

Calcular el factorial de un número entero positivo.

**Conceptos:**

- Acumuladores
- Multiplicación repetitiva
- Iteraciones

---

### B04 - Serie Fibonacci

**Descripción:**

Generar los primeros N términos de la serie Fibonacci.

**Conceptos:**

- Variables auxiliares
- Secuencias numéricas
- Bucles

---

### B05 - Contar Números Positivos

**Descripción:**

Permitir el ingreso de números hasta que se ingrese un valor centinela y contabilizar los números positivos ingresados.

**Conceptos:**

- Bucle Mientras
- Condicionales
- Contadores
- Centinelas

---

## Aprendizajes Esperados

Al finalizar esta sección seré capaz de:

- Automatizar tareas repetitivas.
- Utilizar contadores y acumuladores.
- Diseñar algoritmos iterativos.
- Resolver problemas mediante ciclos controlados.
- Generar secuencias y procesar conjuntos de datos.
